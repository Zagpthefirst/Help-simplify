Quadratic roots:

  • ( c 1 a 2 − a 1 c 2 ) 2 = ( b 1 a 2 − a 1 b 2 ) ⁢ ( c 1 b 2 − c 2 b 1 )  for common root in quadratic
  • Roots of a quadratic are greater than a value ' k ' if: a ⁢ f ⁡ ( k ) > 0 , − b 2 ⁢ a > k , D ≥ 0
  • Roots of a quadratic are lesser than a value ' k ' if: a ⁢ f ⁡ ( k ) > 0 , − b 2 ⁢ a < k , D ≥ 0
  • Roots are on either side of a value ' k ' if: a ⁢ f ⁡ ( k ) < 0
  • Exactly one root of a quadratic is between 2 values ' k 1 ' ,  ' k 2 ' if: f ⁡ ( k 1 ) f ⁡ ( k 2 ) < 0
  • Both roots of a quadratic are between 2 values ' k 1 ' , ' k 2 ' if: a ⁢ f ⁡ ( k 1 ) > 0 , a f ⁡ ( k 2 ) > 0 , k 1 < − b 2 ⁢ a < k 2 , D ≥ 0
  • If one root is bigger than a smaller value ' k 1 ' and other root is larger than a bigger value ' k 2 ' then: a ⁢ f ⁡ ( k 1 ) < 0 , a ⁢ f ⁡ ( k 2 ) < 0
    In this case, both 'k1' and 'k2' are between the roots

Means of mth power:

If m < 0 or m > 1:

a 1 m + a 2 m + a 3 m + ... + a n m n > ( a 1 + a 2 + a 3 + ... + a n n ) m

If 0 < m < 1:

a 1 m + a 2 m + a 3 m + ... + a n m n < ( a 1 + a 2 + a 3 + ... + a n n ) m

If m = 0 or m = 1 or a1 = a2 = a3 = ... = an:

a 1 m + a 2 m + a 3 m + ... + a n m n = ( a 1 + a 2 + a 3 + ... + a n n ) m

Permutations & combinations under restrictions:

Particular objects are separated:

n objects in total arranged such that r objects must be separated number of permutations  = P r n − r + 1 * ( n − r + 1 ) !

Particular objects are together:

n  objects in total arranged so that  a 1  objects of  1 st  type , a 2  objects of  2 nd  type , ... , a k  objects of  k th  type are together number of permutations = ( n − [ a 1 + a 2 + a 3 + ... + a k ] + k ) ! * ( a 1 ) ! * ( a 2 ) ! * ... * ( a k ) !
This is divided by ∏(number of identical elements of ith type)! in case of identical elements

Circular permutations:

Number of circular permutations = number of linear permutations n , n  is number of seats Number of clockwise permutations = number of anti-clockwise permutations = number of circular permutations 2

Selection from identical objects:

Number of ways of selecting  r  objects from  n identical objects = 1 Number of ways of selecting any number of objects from  n  identical objects = n + 1 Number of ways of selecting at least  1  object from  n  identical objects = n
