Points:

  • The term between any 2 terms in a AP/GP/HP is the mean(AM/GM/HM respectively) of the 2 terms
  • If i,j are independent variables in ∑∑(Ti*Tj) then, ∑∑(Ti*Tj) = (ΣTi)(ΣTj), this can be generalized to any number of independent variables
  • If the general term can be expressed as Tn = ±(V(n)−V(n−1)) then the series is telescopic i.e., ΣTn = ±(V(n)−V(0))
  • For Tn = V(n)−V(n−a), ΣTn = V(n)+V(n−1)+...+V(n−a+1)−[V(0)+V(−1)+...+V(−a+1)]

Trivia:

  • ∅ represents an empty set
  • Set minus can be represented as either ∖ or −
  • If a polynomial appear to have more roots than its degree then all the coefficients are 0
  • When differences of consecutive terms form an AP (in a QP) then, the general term is given by Tn = an2 + bn + c
  • When differences of consecutive terms form a GP then, the general term is given by Tn = arn + b
  • And rule is used to find number of ways to perform one task and another
  • Or rule is used to find number of ways to perform one task xorOne or the other but not both another
  • ∏ is product of n terms in a sequence and ∑ represents sum of n terms in a sequence
