Conditions for stuff:

Quadratic roots:

  • (c1a2−a1c2)2 = (b1a2−a2b1)(c1b2−b1c2) for common root in quadratic
  • Roots of a quadratic are greater than a value if aƒ(k) > 0 , −b/2a > k , D ≥ 0
  • Roots of a quadratic are lesser than a value if aƒ(k) > 0 , −b/2a < k , D ≥ 0
  • Roots are on either side of a value if aƒ(k) < 0
  • Exactly one root of a quadratic is between 2 values if ƒ(k1)ƒ(k2) < 0
  • Both roots of a quadratic are between 2 values if aƒ(k1) > 0 , aƒ(k2) > 0 , k1 < −b/2a < k2 , D ≥ 0
  • If one root is bigger than the smaller value and other root is larger than the bigger value then aƒ(k1) < 0 , aƒ(k2) < 0

Loci in Argand plane:

    1. Point inside a line segment ≡ |z−z1| + |z−z2| = |z1+z2|
    2. Ellipse ≡ |z−z1| + |z−z2| = k , k > |z1−z2| , e = |z1−z2|/k
    3. Ray emitting from one side of a line segment ≡ |z−z1|−|z−z2| = |z1−z2|
    4. Point on the line but not on the line segment ≡ |(|z−z1|−|z−z2|)| = |z1−z2|
    5. Hyperbola ≡ |(|z−z1|−|z−z2|)| = k , k<|z1−z2| , e = |z1−z2|/k
    1. Apollonius circle ≡ |z−z1|/|z−z2| = k , k ≠ 0,1
    2. Circle ≡ |z−z1| = r
    3. Circle with diameter points ≡ |z−z1|2 + |z−z2|2 = |z1−z2|2
    4. Circle with diameter points ≡ (z−z1)(z̅−z̅2)+(z−z2)(z̅−z̅1) = 0
    1. Line ≡ arg(z−z1) = θ
    2. Line with 2 points ≡ a̅z + z̅a + b = 0 , a = i(z2−z1) , b = i(z1z̅2−z̅1z2) , slope = −re(a)/im(a)
  • Perpendicular bisector ≡ |z−z1| = |z−z2|
  • Arc ≡ arg([z−z1]/[z−z2]) = α
  • Concyclic points ≡ [(z4−z1)/(z2−z1)][(z2−z3)/(z4−z3)] is real
  • Equilateral triangle ≡ 1/(z1−z2) + 1/(z2−z3) + 1/(z3−z1) = 0
  • Section formula for complex number ≡ (mz2+nz1)/(m+n) for internal and (mz2−nz1)/(m−n) for external

Means of mth power:

If m < 0 or m > 1:

(a1m + a2m + a3m + ... + anm)/n > [(a1 + a2 + a3 + ... + an)/n]m

If 0 < m < 1:

(a1m + a2m + a3m + ... + anm)/n < [(a1 + a2 + a3 + ... + an)/n]m

If m = 0 or m = 1 or a1 = a2 = a3m = ... = an:

(a1m + a2m + a3m + ... + anm)/n = [(a1 + a2 + a3 + ... + an)/n]m

Permutations & combinations under restrictions:

Particular objects are separated:

n objects in total arranged such that r objects must be separated
number of permutations = n-r+1Pr * (n-r+1)!

Particular objects are together:

n objects in total arranged so that a1 objects of 1st type, a2 objects of 2nd type,...,ak objects of kth type are together
number of permutations = (n-[a1 + a2 + a3 + ... + ak] + k) * (a1)!*(a2)!*...*(ak)!{divided by ∏(number of identical elements of ith type)! in case of identical elements}

Circular permutations:

Number of circular permutations = (number of linear permutations)/n, n is number of seats
Number of clockwise permutations = number of anti-clockwise permutations = (number of circular permutations)/2

Selection from identical objects:

Number of ways of selecting r objects from n identical objects = 1
Number of ways of selecting any number of objects from n identical objects = n+1
Number of ways of selecting at least 1 object from n identical objects = n

Divisors of a number:

Prime factorization of N = ∏[piai]
Number of divisors of N = ∏(ai + 1)
Sum of divisors of N = ∏([piai+1 − 1]/[p1 − 1])
Number of ways of writing N as a product of 2 natural numbersas unordered pair, (a,b) is same as (b,a) =
